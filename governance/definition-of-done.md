# Definition of Done

## General

- [ ] 目標與交付物一致
- [ ] 必要輸入完整
- [ ] 假設已標示
- [ ] 重要事實有來源
- [ ] 數字已交叉檢查
- [ ] 沒有敏感資訊外洩
- [ ] 風險等級已判定
- [ ] 人工核准點已完成
- [ ] 版本與 Owner 已記錄
- [ ] 未完成事項已明示

## Skill

- [ ] 一個 Skill 只處理一種工作
- [ ] description 清楚
- [ ] 有 Use / Do Not Use
- [ ] 有輸入契約
- [ ] 有輸出契約
- [ ] 有步驟與品質檢查
- [ ] 有失敗處理
- [ ] 有三類測試案例

## Workflow

- [ ] 有觸發條件
- [ ] 有 Owner
- [ ] 有步驟順序
- [ ] 有依賴關係
- [ ] 有交接契約
- [ ] 有品質閘門
- [ ] 有停止條件
- [ ] 有人工核准點
